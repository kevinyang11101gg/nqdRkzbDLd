# 项目简介

## 核心功能
- 登录注册
- 菜品信息推荐
- 菜品信息详情
- 收藏
- 评论
- 点赞
- 加入购物车
- 下单
- 用户管理
- 菜品管理
- 菜品分类管理
- 订单管理
- 个人中心

## 适用场景
- 快餐店、餐厅等餐饮服务场所
- 企业内部食堂
- 在线外卖平台

## 技术特点
- 采用Django框架进行后端开发，确保系统稳定性和安全性
- 使用MySQL数据库存储数据，支持大规模数据存储和查询
- 前端采用Vue.js，提供良好的用户体验和交互效果
